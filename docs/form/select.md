> Attributes

参数 | 说明 | 类型 | 可选值 | 默认值
---|---|---|---|---
list | 列表, 比传参数 | array | - | -
model | 选中值 | string | - | -
defaultText | placeholder | string | - | -
size | 大小 | string | small, large, - | -
disabled | 是否禁用 | boolean | - | false
style | 自定义样式 | string | - | -
placement | 弹窗的展开方向 | string | bottom 和 top | -
