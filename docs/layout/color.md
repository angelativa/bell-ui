为了避免视觉传达差异，使用一套特定的调色板来规定颜色，为你所搭建的产品提供一致的外观视觉感受。

> 主色

<html>
    <style type="text/css">
        .card {
            display: inline-block;
            margin: 10px;
            color: #fff;
            padding: 10px;
            width: 200px;
            border-radius: 4px;
            line-height: 30px;
            font-size: 18px;
        }
    </style>
    <div class="card" style="background-color: #1c90f3;">
        Primary
        <div style="font-size: 14px;">#1c90f3</div>
    </div>
</html>
